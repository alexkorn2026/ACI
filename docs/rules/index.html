ACI-Regeldokumentation

Diese Seiten beschreiben alle von ACI (Automated Code Inspection) ausgewerteten Regeln – je Regelsatz und je einzelner Regel. ACI prüft Oracle-PL/SQL- und PostgreSQL-PL/pgSQL-Code in drei Regelbereichen: Sicherheit, Coding Guidelines und MITRE-ATT&CK-Angriffsindikatoren.

Die Seiten werden aus den JSON-Regeldateien unter aci/rules/ erzeugt und spiegeln den ausgelieferten Regelstand wider.
